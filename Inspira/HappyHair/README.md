# HappyHair

## Overzicht

Website voor een kapsalon met focus op een moderne uitstraling, gebruiksvriendelijke navigatie en duidelijke presentatie van de aangeboden diensten. De website biedt bezoekers snel toegang tot informatie over behandelingen, contactgegevens en praktische informatie.

## Mijn bijdrage

- Structuur en opbouw van de website
- Vormgeving en lay-out
- Responsive uitwerking
- Uitwerken van informatiepagina's
- Optimalisatie van de gebruikerservaring
- Presentatie van diensten en contactinformatie

## Wat ik geleerd heb

Tijdens dit project leerde ik hoe een website voor een lokale dienstverlener wordt opgebouwd met aandacht voor zowel uitstraling als gebruiksgemak. Ik kreeg ervaring met het vertalen van klantverwachtingen naar een overzichtelijke website en het presenteren van informatie op een manier die bezoekers snel naar de juiste inhoud begeleidt.
