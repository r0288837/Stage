# CleanTeam Kempen

## Overzicht

Website voor een professioneel schoonmaakbedrijf actief in de Kempen. De website werd ontworpen met een focus op vertrouwen, dienstverlening en een duidelijke presentatie van de aangeboden diensten.

## Mijn bijdrage

- Structuur en opbouw van de website
- Vormgeving en lay-out
- Responsive uitwerking
- Uitwerken van dienstpagina's
- Klantgerichte informatiearchitectuur

## Wat ik geleerd heb

Tijdens dit project leerde ik hoe een professionele bedrijfswebsite wordt opgebouwd voor een lokale dienstverlener. Hierbij werd veel aandacht besteed aan gebruiksvriendelijkheid, duidelijke navigatie en het overzichtelijk presenteren van informatie aan potentiële klanten.
