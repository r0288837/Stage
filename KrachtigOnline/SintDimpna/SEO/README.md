# KSAV Sint-Dimpna SEO Optimalisatie

## Overzicht

SEO-onderzoek en optimalisatiewerkzaamheden uitgevoerd voor de website van KSAV Sint-Dimpna. Het doel was om de vindbaarheid van de website te verbeteren door technische analyse, contentoptimalisatie en het identificeren van verbeterpunten.

## Mijn bijdrage

- Uitvoeren van SEO-analyses
- Crawlen van de website met Screaming Frog
- Identificeren van technische SEO-problemen
- Opstellen van optimalisatiechecklists
- Formuleren van aanbevelingen voor verbeteringen
- Analyseren van pagina-structuur en metadata
- Onderzoeken van zoekmachinevriendelijkheid van de website
- Documenteren van bevindingen en verbeterpunten

## Wat ik geleerd heb

Tijdens dit project leerde ik hoe zoekmachines websites analyseren en indexeren. Ik kreeg ervaring met het gebruik van professionele tools zoals Screaming Frog en leerde technische en inhoudelijke SEO-factoren beoordelen.

Daarnaast ontwikkelde ik inzicht in metadata, pagina-structuren, interne linken, indexatieproblemen en de impact van technische optimalisaties op de online zichtbaarheid van een website.
