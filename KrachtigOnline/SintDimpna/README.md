# KSAV Sint-Dimpna

## Overzicht

Website voor voetbalclub KSAV Sint-Dimpna waarbij competitiegegevens, uitslagen, rangschikkingen en wedstrijdinformatie toegankelijk worden gemaakt voor spelers, ouders, supporters en bezoekers.

## Mijn bijdrage

- Integratie van externe voetbalgegevens via API-koppelingen
- Ontwikkelen van een WordPress shortcode-oplossing
- Structureren van competitie-, uitslagen- en rangschikkingspagina's
- Implementeren van fallback-oplossingen voor gegevensweergave
- Testen en optimaliseren van de gegevensverwerking
- Integratie binnen de bestaande WordPress-omgeving

## Wat ik geleerd heb

Tijdens dit project leerde ik hoe externe databronnen kunnen worden geïntegreerd binnen een bestaande website. Ik deed ervaring op met API-verwerking, het weergeven van dynamische gegevens in WordPress en het ontwikkelen van oplossingen die blijven functioneren wanneer externe diensten tijdelijk niet beschikbaar zijn. Daarnaast kreeg ik meer inzicht in het testen en valideren van gegevensstromen binnen een webapplicatie.
